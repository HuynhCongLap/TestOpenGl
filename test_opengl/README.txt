Ce dossier est copié-collé depuis le répertoire d'exemples de Qt.
Quelques changements y ont ensuite été apportés.
Notamment, le fichier "logo.h" contient une macro "MODDED" qui, selon si elle est définie ou pas, lance la version d'origine ou la version modifiable du projet.

Le but du test est d'arriver à un résultat similaire à ce qui est présenté dans la vidéo "target.mp4" : remplacer le modèle statique de l'exemple original par un modèle simple animé.

Pour cela, la modification des fichiers "vertexShaderSourceMODDED.glsl" et "fragmentShaderSourceMODDED.glsl" peut être suffisante.
Bien entendu, arriver à un résultat meilleur ou même différent (tant qu'il reste joli/intéressant/impressionnant) est accepté voire encouragé.
